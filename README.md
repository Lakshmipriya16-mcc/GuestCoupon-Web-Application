# Guest Coupon Application

## Project Overview
The Guest Coupon Application allows employees to raise and manage food coupons for visitors from external organizations. It streamlines request submission, approval, notifications, and tracking, providing a seamless digital workflow.

## Features
- Employee login & registration
- Submit coupon requests with guest details and attachments
- Manager approval/rejection with comments
- Email notifications for request updates
- Request history and status tracking
- Reporting for managers

## Technologies Used
- **Front-end:** HTML, CSS, JavaScript  
- **Back-end:** PHP  
- **Database:** MySQL (phpMyAdmin)  
- **Local Server:** XAMPP
